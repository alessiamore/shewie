# 💰 Expense Tracker

Keeping track of everyday spending doesn't have to be complicated.

This small Python application stores expenses, groups them by category and prints a spending summary.

---

## Quick Demo

```
Expense Tracker

Coffee          Food        $4.50
Internet        Utilities   $35.00
Book            Education   $18.00

----------------------------

Total expenses : $57.50

Food           $4.50
Utilities      $35.00
Education      $18.00
```

---

## Files

| Name | Description |
|------|-------------|
| app.py | Entry point |
| ledger.py | Stores expenses |
| transaction.py | Expense model |
| summary.py | Calculates totals |
| categories.py | Category reports |
| printer.py | Console output |
| demo_data.py | Sample expenses |

---

## Running

```bash
python app.py
```
